# Insurance API (`services/insurance-api`)

Second Salanor product on the shared platform (Stage 11 scaffold).

- Health: `GET /health`
- Console: `GET /v1/insurance/console/overview` (requires Salanor ID session)

Default port: `8092` (`INSURANCE_API_PORT`).

Console UI: `http://localhost:3000/insurance` (routes only — no separate GitHub repo).
